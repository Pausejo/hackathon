## How to run

1. Clone the repository:

```bash
git clone https://github.com/pausejo/hackathon.git
cd hackathon
```

2. Install the required dependencies and run backend:

```bash
pip install -r requirements.txt
python main.py
```

4. Run the frontend:

```bash
cd front
npm install
npm run dev
```
